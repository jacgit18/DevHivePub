### Key Topics in Data Analysis

1. **Web Scraping**:
   - The process of automatically extracting data from websites.

2. **Data Cleaning**:
   - Preparing raw data by handling missing values, correcting errors, and formatting data for analysis.

3. **Data Visualization**:
   - Creating graphical representations of data to identify patterns, trends, and insights.

4. **Exploratory Data Analysis (EDA)**:
   - Analyzing data sets to summarize their main characteristics, often using visual methods. This involves:
     - Asking questions
     - Performing initial analysis
     - Validating findings

5. **Sentiment Analysis**:
   - A technique in natural language processing (NLP) for determining the sentiment of textual data, classifying it as neutral, positive, or negative.
