**Snake case** is a naming convention used in programming and coding where:  
  
- **Words are written in lowercase**  
- **Separated by underscores (_)**  
  
### Examples:  
- `user_name`  
- `total_amount_calculator`  
- `is_valid_input`  
  
### Key Uses:  
1. **Variable & Function Naming** (common in Python, Ruby, etc.)  
```python  
def calculate_total_price():  
tax_rate = 0.05  
```  
2. **File Naming** (e.g., `config_settings.json`)  
3. **Database Column Names** (e.g., `created_at`)  
  
### Contrast With Other Naming Styles:  
- **CamelCase**: `userName`  
- **PascalCase**: `UserName`  
- **Kebab Case**: `user-name` (used in URLs/filenames)  
  
### Why Use Snake Case?  
- Improves readability for multi-word identifiers.  
- Standard in languages like Python ([PEP 8](https://peps.python.org/pep-0008/)).
